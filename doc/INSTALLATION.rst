Installation

Requirements
============

These requirements should be installed in a Python virtual environment::

    1. cherrypy
    
    #. lxml (version 2.2.8)
    
    #. psycopg2 (version 2.2.2)
    
    #. geopy (check this!)
    
    #. mox
    
User Account
============

Create a system user for the web application to run as.  'pyuser' is a
reasonable account name.
